Thank you for installing Pillz.

First step

Open a new terminal and run the following command to get check the installation:

  $ pillz --version

Uninstall Pillz

Run the following command to uninstall Pillz.

  $ sudo bash /Library/__PRODUCT__/__VERSION__/uninstall.sh
